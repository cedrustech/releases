Latest Base Version - v3.23.9

New Features and Enhancements

New 1. Implemented GST Functionality for Invoices

»» Now, you can easily apply GST rates of 5%, 12% or 18% to your invoices, and the corresponding GST amount will be automatically calculated and added to the total amount.

New 2. Icon for Mode of Transactions

»» We've enhanced the user experience by adding icons for each mode of transaction. When you select a mode of transaction, such as "cash," you will see a Cash icon, making it even more intuitive to identify your chosen mode.

New 3. Total Amount Calculation in MoT Fields

»» Introducing automation that dynamically reflects the total amount in a specific field based on the selected mode of transaction. For example, if you choose the mode of transaction as "Cash," the total amount will be displayed in the Cash Amount field, while all other fields will show zero. This feature simplifies reporting by allowing you to segregate billing calculations based on the mode of transaction.

What's Changed/Fixed

Changed 1. Additional Access for Incharge Profile

»» We have expanded the accessibility of the Incharge profile to provide them with additional permissions and functionalities.

Fixed 2. Resolved Stock Name Error (STRING_TOO_LONG)

»» We have fixed a critical issue related to stock names exceeding the character limit, which was causing errors. Now, your data can be managed seamlessly without any constraints.

Improved 3. Streamlined Invoice Creation

»» To improve user experience and reduce confusion, we have removed the "Add Item" button from the Invoice module. You can use drop-down 'New' button to add item. This simplifies the process and enhances usability.

Changed 4. Removed Picklist Value - MoT

»» We have deleted one value from the "Mode of Transaction" picklist to ensure data consistency and accuracy in your records.

Changed 5. Enhanced Invoice Status Icon

»» The "Paid" status icon in invoices has been redesigned for better clarity and visual representation, making it easier to identify paid invoices at a glance.

Fixed 6. Fixed Quick Actions Visibility Issue

»» Quick actions for "View Invoice" and "Mark Paid" are now visible as intended. Users can efficiently access and manage invoices without any hindrance.

Functional Updates

Changed 1. Gatepass and Outward Object Enhancements

»» We have made functional modifications to the Gatepass and Outward objects, enhancing their capabilities and improving overall workflow efficiency.

Changed 2. Automation Adjustment - Add to Sale Field (Stock)

»» We have removed the automation for the "Add to Sale" field in the Stock object. Additionally, the default value for this field has been changed to TRUE, simplifying data entry and reducing manual effort.

New Features and Enhancements

New 1. Analog and Digital Clock on Home Page

»» Keep track of time conveniently right from your home page with the newly added analog and digital clocks.

New 2. Quantity Details Table on Stock Record Page

»» Easily view and manage quantity details for your stock records like Available Quantity, Sold Out Quantity, In Cart Quantity and Outward Quantity on the Stock Record Page.

New 3. Discount Functionality for Invoice

»» Streamline your billing process with the addition of a discount functionality for invoices, where in you can add discount percentage while adding sale item in invoice bill.

New 4. Button to Pay and View Invoice

»» Simplify invoice management by providing a one-click option to pay and view invoices directly.

New 5. Gatepass Functionality with Automations

»» Introducing a comprehensive gatepass functionality with automation features to enhance security and control. This feature can be use for outward stock operation management.

New 6. Button to Close and View Gatepass

»» Quickly close and view gatepasses with the new button for added convenience.

What's Changed/Fixed

Changed 1. Application Logo and Theme

»» Enjoy a fresh look with updated application branding and theme.

Fixed 2. Auto-Capital Functionality Issue

»» Addressed and resolved the auto-capital functionality issue for smoother data entry.

Improved 3. Apex Code Coverage Result - 100%

»» Enhancements have been made to ensure that our app maintains 100% Apex code coverage for optimal performance and reliability.
